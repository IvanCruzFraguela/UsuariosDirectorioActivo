# UsuariosDirectorioActivo
La idea original es crear un fichero de texto con los nombres y nombres de usuario almacenados en un Directorio Activo y, de momento es lo que tenemos.

El programa recibe dos parámetros:
El primero el la entrada de directorio en donde se encuentren los usuarios a meter en el fichero. Es obligatorio ponerlo.
El segundo es el fichero donde se guardarán esos datos.
